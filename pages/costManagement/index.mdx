# Cost Management

The Cost Management page provides visibility into your cloud spending and identifies opportunities to reduce costs. It is available as a tab within the **Security and Compliance** assessment view.

To access Cost Management, navigate to **Security and Compliance**, select a cloud environment and scan date, then click the **Cost Management** tab.

## Supported Providers

Cost Management is available for the following cloud providers:

- [**Azure**](/costManagement/azure) — Combines billing data from Azure Cost Management with optimization recommendations from Azure Advisor and the ASecureCloud assessment engine.

Select your cloud provider above to view the full documentation.
